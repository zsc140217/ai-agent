# 某某科技有限公司差旅报销制度

## 1. 城市分类
- **一类城市**：北京、上海、广州、深圳、杭州、成都。
- **二类城市**：其他省会城市。

## 2. 交通标准
- **高铁**：一类及二类城市均可乘坐二等座。
- **飞机**：单程飞行时间超过 4 小时或单程票价低于高铁二等座时，可申请经济舱。

## 3. 住宿报销限额 (每人每天)
| 城市级别 | 报销上限 | 备注 |
| :--- | :--- | :--- |
| 一类城市 | 500 元 | 杭州包含在内 |
| 二类城市 | 350 元 | |

## 4. 补贴标准
- 伙食补助：100 元/天。
- 市内交通补助：80 元/天（若已产生打车费报销，则取消该补助）。
