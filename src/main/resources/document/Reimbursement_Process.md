# 差旅费用报销流程指南

---

## 1. 报销流程总览

```
出差申请 → 出差执行 → 费用整理 → 提交报销 → 审批流程 → 财务打款
```

**时间要求**：出差结束后 **7 个工作日内** 必须提交报销申请

---

## 2. 出差前准备

### 2.1 填写出差申请单

**必填信息：**
- 出差人姓名、部门、工号
- 出差目的（客户拜访/项目实施/会议参加等）
- 出差地点（具体城市和客户名称）
- 出差时间（起止日期）
- 预计费用（交通、住宿、补贴）
- 同行人员（如有）

**申请单模板下载：**
- OA 系统：行政管理 → 差旅管理 → 出差申请单
- 或联系行政部获取

### 2.2 审批流程

| 出差天数 | 审批流程 |
| :--- | :--- |
| 1-3 天 | 部门经理审批 |
| 4-7 天 | 部门经理 → 部门总监审批 |
| 8 天以上 | 部门经理 → 部门总监 → 人力资源部审批 |
| 国际出差 | 部门经理 → 部门总监 → 人力资源部 → 总经理审批 |

**审批时间：**
- 国内出差：提前 3 个工作日申请
- 国际出差：提前 10 个工作日申请（需办理签证）

### 2.3 预订交通和酒店

**推荐方式：**
1. **公司统一预订**（推荐）
   - 联系行政部：0571-8888-1235
   - 可享受协议价，自动开具公司抬头发票
   
2. **自行预订**
   - 通过携程、美团等平台
   - 注意选择可开具增值税专用发票的商家
   - 发票抬头必须是公司全称

**注意事项：**
- 优先选择协议酒店（见《协议酒店清单》）
- 高铁票提前 7 天预订可享受更优惠价格
- 机票提前 14 天预订可节省 20-30% 费用

---

## 3. 出差中注意事项

### 3.1 保留所有凭证

**必须保留的凭证：**
- ✅ 交通费：高铁票、机票行程单、出租车发票、地铁票
- ✅ 住宿费：酒店增值税专用发票
- ✅ 餐饮费：正规餐饮发票（客户招待需注明用餐人员）
- ✅ 其他费用：停车费、过路费、会议费等

**发票要求：**
- 必须是正规发票（增值税普通发票或专用发票）
- 发票抬头：某某科技有限公司
- 税号：91330000XXXXXXXXXX
- 电子发票需打印纸质版

### 3.2 拍照备份

**建议操作：**
- 每天晚上整理当天的发票
- 用手机拍照备份（防止丢失）
- 按时间顺序整理

### 3.3 记录工作内容

**出差日志：**
- 每天记录拜访的客户、会议内容
- 记录工作成果和下一步计划
- 报销时需要附上出差总结

---

## 4. 报销材料准备

### 4.1 必备材料清单

1. **《差旅费报销单》**（原件）
   - 填写完整，不得涂改
   - 金额大小写必须一致
   
2. **《出差申请单》**（复印件）
   - 需有审批人签字
   
3. **所有发票**（原件）
   - 按时间顺序粘贴在 A4 纸上
   - 每张纸不超过 10 张发票
   
4. **出差总结**（Word 文档）
   - 简要说明出差目的、工作内容、成果
   - 300 字以上

5. **客户招待费用**（如有）
   - 需单独填写《招待费报销单》
   - 附客户名单和招待事由

### 4.2 发票粘贴规范

**粘贴要求：**
- 使用 A4 纸，横向粘贴
- 从左到右、从上到下依次粘贴
- 不要重叠，确保每张发票清晰可见
- 在发票旁边标注日期和用途

**示例：**
```
[高铁票]     [出租车发票]   [餐饮发票]
3月1日       3月1日         3月1日
杭州→上海    上海市内交通    午餐
```

## 5. 提交报销申请

### 5.1 线上提交（推荐）

**操作步骤：**
1. 登录 OA 系统
2. 进入"财务管理" → "费用报销"
3. 选择"差旅费报销"
4. 填写报销单信息
5. 上传发票扫描件
6. 提交审批

**优点：**
- 审批流程自动流转
- 可随时查看审批进度
- 到账更快

### 5.2 线下提交

**操作步骤：**
1. 打印《差旅费报销单》
2. 粘贴所有发票
3. 提交到财务部（行政楼 3 楼）
4. 领取报销单回执

**提交时间：**
- 周一至周五 9:00-17:00
- 午休时间 12:00-13:00 不受理

## 6. 审批流程

### 6.1 审批层级

| 报销金额 | 审批流程 |
| :--- | :--- |
| 5000 元以下 | 部门经理 → 财务审核 |
| 5000-10000 元 | 部门经理 → 部门总监 → 财务审核 |
| 10000 元以上 | 部门经理 → 部门总监 → 财务总监 → 财务审核 |

### 6.2 审批时间

- **部门审批**：1-2 个工作日
- **财务审核**：2-3 个工作日
- **总计**：3-5 个工作日

### 6.3 审批结果查询

**线上查询：**
- OA 系统 → 我的申请 → 查看进度

**线下查询：**
- 致电财务部：0571-8888-1234

---

## 7. 财务打款

### 7.1 打款时间

- 审批通过后 **5 个工作日内** 到账
- 每月 25 日后提交的报销，次月 5 日前到账

### 7.2 打款方式

- 直接打入员工工资卡
- 短信通知到账

### 7.3 到账查询

- 查看工资卡余额
- 或致电财务部查询

## 8. 常见问题处理

### 8.1 发票丢失怎么办？

**解决方案：**
1. **高铁票/机票**：
   - 联系 12306 或航空公司补打行程单
   - 提供订单号和身份证号
   
2. **酒店发票**：
   - 联系酒店前台补开发票
   - 提供入住日期和房间号
   
3. **出租车发票**：
   - 无法补开，只能自行承担

**预防措施：**
- 每天拍照备份所有发票
- 使用专用文件袋保管发票

### 8.2 发票抬头错误怎么办？

**解决方案：**
- 联系开票方重新开具
- 原发票需要作废或红冲
- 无法重开的发票不予报销

**正确抬头：**
- 名称：某某科技有限公司
- 税号：91330000XXXXXXXXXX

### 8.3 超标准报销怎么办？

**解决方案：**
1. **提前申请**：
   - 出差前填写《超标准申请单》
   - 说明原因（如展会期间酒店涨价）
   - 部门总监审批
   
2. **事后申请**：
   - 报销时附上情况说明
   - 部门总监签字确认
   - 超出部分可能不予报销

### 8.4 报销被退回怎么办？

**常见退回原因：**
- 发票不合规（无公司抬头、非正规发票）
- 金额计算错误
- 缺少必要材料（出差申请单、出差总结）
- 超出报销标准且未提前申请

**处理方式：**
- 查看退回原因
- 补充或修改材料
- 重新提交

## 9. 特殊情况处理

### 9.1 紧急出差

**定义：**
- 临时接到任务，无法提前申请

**处理方式：**
- 先出差，后补申请
- 出差后 2 个工作日内补交《出差申请单》
- 说明紧急原因

### 9.2 客户招待费用

**报销要求：**
- 单独填写《招待费报销单》
- 附客户名单（姓名、公司、职位）
- 说明招待事由和业务关系
- 部门总监审批

**报销标准：**
- 人均 200 元以内
- 不得购买烟酒

### 9.3 国际出差

**额外材料：**
- 护照复印件
- 签证复印件
- 国际机票行程单
- 酒店英文发票（需翻译）

**汇率计算：**
- 按出差当天中国银行外汇牌价中间价计算
- 或提供信用卡账单
